Долгосрочное задание  
Вам предоставлена таблиц в формате csv.
Необходимо:
1.	Загрузить таблицу в СУБД PostgreSQL
2.	Привести БД к третьей нормальной форме
Комментарий:
●	Чтобы загрузить файл csv, необходимо предварительно создать таблицу со 
всеми столбцами как в файле. Не забудьте правильно указать типы столбцов.
●	Таблица состоит из сущностей staff, order, client и cargo. Одна строка - это один 
заказ. Заказ состоит из order, client, cargo. Получается, что один заказ могут 
выполнять несколько сотрудников.

Задание сдается в виде дампа БД и скриптов на SQL, которые вы использовали в ходе 
решения задачи.

Легенда таблицы:
Долгое время администратор логистической компании записывал заказы в одну 
таблицу. Таким образом в одну таблицу попали данные по сотрудникам (staff), заказам 
(order), грузам (c/cargo) и клиентам (cli). Разные администраторы считали 
необходимыми разные данные, так что в таблице есть пробелы.

Особые поля:
●	У заказов есть поле order_list, которое состоит из id грузов.
●	У клиентов есть секретное слово, пин, и токен, для различных способов 
получения заказа. 

Описание столбцов: 

staf_name - имя и фамилия исполнителя заказа
staff_age - возраст исполнителя заказа
staff_id - уникальный идентификатор сотрудника
staff_lang - основной язык сотрудника 
 
order_pk - primary key заказа 
order_address - адрес доставки 
order_country - страна доставки 
order_company - компания заказчик 
order_price - стоимость заказа 
order_dt - дата заказа 
order_list - список id предметов в посылке 
 
cli_name - имя менеджера компании заказчика 
cli_email - почта менеджера компании заказчика 
cli_phone - телефон менеджера компании заказчика 
cli_secret - токен для доступа к панели заказа на сайте

c_token - токен для транспортной компании для доступа к панели груза на сайте 
c_pin - код для получения груза 
c_gen - поколение груза 
c_type - тип груза

Заказ определяется сущностями order, client, cargo.
Один заказ может выполнять несколько сотрудников.
Заказ получает человек по указанному адресу, при условии, что назвал пин код.
